About My Application

The core basis of my Application is a Loan
Repayment calculator.

This application will take the Client/Customer - Salary, Amount to borrow, Interest
and the total term of loan and calculate the repayments.
This will help the user determine their borrowing capacity.

To implement the App, All you must do is click the run button and follow the written prompts,
the output will be shown in the console and written to .txt file
